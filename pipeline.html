End-to-End Pipeline

The complete intended path from real-world input to display — with the Adaptive Voxel World as the single source of truth, and AI only improving observations and caches.

Authoritative flow Ingest Voxel world Simulation Compiler Gaussians Enhancement Display Background Status Roadmap Updates

Goal: one coherent pipeline, not a pile of tools

Reality becomes evidence in an Adaptive Voxel World. Simulation changes that world. The World Compiler emits derived artifacts (including Gaussians). Heavy AI may improve the Gaussian cache when compute is free. A rasterizer produces a frame packet every frame. Light local AI and optional full-frame AI enhance the image. Display never owns authority.

Authoritative Data Flow

Reality ↓ Ingestion & Normalization ↓ Registration / Evidence ↓ Adaptive Voxel World ←── SINGLE SOURCE OF TRUTH ↓ Simulation ↓ World Compiler ↓ Gaussian Representation ↓ Background Gaussian Optimization (heavy AI, cached, not every frame) ↓ Enhanced Gaussian Cache ↓ Gaussian Rasterizer → GaussianImagePacket ↓ Real-Time Local Enhancement (light AI, every frame, optional) ↓ Optional Full-Frame Post Process ↓ Display (Desktop · Browser · VR · AR · Mobile · Cloud)

Background improvement can update the Gaussian cache without rebuilding the voxel world, re-running simulation, re-importing data, or interrupting rendering.

Background Gaussian Optimization

Optional Heavy AI

FLUX · HiDream · future foundation models — improve the Gaussian representation offline or on spare capacity. They do not run in the real-time display path. Registered as guarded plugins until GPU weights exist.

Enhanced Gaussian Cache (V19) — optimized splat revisions, nothing overwritten, history preserved. Browser / V18 path transparently prefer best_arrays(). Only recompute when necessary.

Real-Time Local Gaussian Enhancement

Runs when enabled. Packet-only — never touches the voxel world. Profiles scale low (passthrough) → medium → high → cloud. Live classical-CV enhancers; AI stages (RealESRGAN / photorealistic / reflection) are registered but guarded until GPU weights exist.

Sharpen Details Better Glass Cleaner Edges Improve Reflections Reduce Splat Artifacts Better Blending Anti-Aliasing Texture Recovery Surface Detail Temporal Stability

→ Enhanced Frame — a photorealism-ward Gaussian image, still an observation. See V18 report and architecture status.

Optional Full-Frame AI Post Process

Operates on the final image only — entirely optional (still goal):

HDR Tone Mapping Atmospheric Effects Bloom Lens Effects Depth of Field Film Grain Stylization Anime Watercolor Night Vision Thermal Accessibility Filters Upscaling DLSS/FSR-style Reconstruction
