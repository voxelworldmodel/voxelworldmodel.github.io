## Abstract

This repository hosts the public project site for **Voxel World Model (VWM)**, an open architecture for representing the world as a living, synchronized voxel simulation. The site presents the platform overview, development milestones, and measured benchmarks for the VWM-native appearance compiler.

---

## Contents

| Path | Description |
|------|-------------|
| `index.html` | Main overview — philosophy, architecture, neural rendering, current status |
| `updates.html` | Milestone reports (appearance compiler v1, v2) |
| `benchmarks.html` | Comparative benchmark tables and runtime proofs |
| `site.css` | Shared stylesheet |

Static assets include illustrative renders of the voxel world, point-cloud view, and voxel-to-photoreal comparison figures.
